transform 
= how to transform/show your element 
there are functions used to set element transformation.
1. rotate 
    rotate(), rotateX(), rotateY(), rotateZ(), rotate3d()
    property : rotate 

2. scale 
    scale(), scaleX(), scaleY(), scaleZ(), scale3d()
    property : scale 

3. skew 
    skew(), skewX(), skewY()

4. translate 
    translate(), translateX(), translateY(), translateZ(), translate3d()
    property : translate 

5. perspective 
    perspective()
    property : perspective

6. matrix 
    matrix(), matrix3d()
